Report KPI V.

 

Selezionando la voce di menu Report KPI V, l’utente visualizza la seguente maschera che permette di generare il report KPI o di generarlo e salvarlo sul SIR.

 

 

·          Mese Inizio: selezionare un valore attraverso il match code per indicare il mese dal quale il report deve iniziare.

 

·          Mese Fine: selezionare un valore attraverso il match code per indicare il mese nel quale il report deve finire.

 

·          Anno: selezionare un valore attraverso il match code per indicare l’anno.

 

 

Selezionare il pulsante per effettuare il download del report.

 

 

Selezionare il pulsante  per inviare al SIR il report.

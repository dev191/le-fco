Download Documenti KPI

 

Selezionando la voce di menu Download Documenti KPI, l’utente visualizza la seguente maschera che permette di effettuare il download dei KPI, KPI SLA Salvati, KPI Inviati al server.

 

 

Selezionare un valore attraverso il match code Tipo Doc per indicare cosa si vuole ricercare.

 

KPI

 

 

·          Edificio: selezionare l’edificio per il quale si vuole effettuare il download dei KPI attraverso il match code.

 

·          Nome Documento: è un campo di ricerca libera in cui inserire parte o tutto il nome del documento che si sta cercando.

 

·          Anno: selezionare un valore attraverso il match code per indicare l’anno.

 

·          Mese: selezionare un valore attraverso il match code per indicare il mese.

 

 

KPI SLA Salvati

 

·          Tipo Doc: selezionare un valore attraverso il match code per indicare cosa si vuole ricercare.

 

 

KPI Inviati al server

 

·          Tipo Doc: selezionare un valore attraverso il match code per indicare cosa si vuole ricercare.

 

 

Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

 

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

 

 

Selezionare il tasto  per visualizzare, nel frame in basso, tutte le richieste che soddisfano i criteri impostati. Per ognuna di esse vengono date le seguenti informazioni di dettaglio:

 

- Download

- Documento

 

Selezionando l’icona   corrispondente al documento, l’utente può effettuare il download del file zip nel quale troverà il KPI

Report Trimestrale

 

Selezionando la voce di menu Report Trimestrale, l’utente visualizza la seguente maschera che permette di generare i report trimestrali.

 

 

Per effettuare la ricerca l'utente deve utilizzare come filtri tutti i campi visualizzati all'interno della maschera.

 

I campi visualizzati sono:

 

·          Edificio: selezionare un edificio attraverso il match code per indicare l’edificio per il quale si vuole il report trimestrale.

 

·          Trimestre: selezionare un trimestre attraverso il match code per indicare il trimestre per il quale si vuole il report.

 

·          Anno: selezionare un anno attraverso il match code per indicare l’anno per il quale si vuole il report trimestrale.

 


Selezionando il pulsante   l’utente visualizza un pop up dove verrà segnalato che si sta effettuando il download del file zip.

 

 

Una volta salvato il file zip sul computer l’utente andrà ad aprirlo e dentro troverà il file excel del relativo report trimestrale.

 

Selezionando il pulsante   l’utente salverà all’interno del SIR il report trimestrale, il quale poi sarà revisionabile nella sezione Registro spedizioni SGA DIE.

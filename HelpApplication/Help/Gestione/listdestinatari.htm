Destinatari SGA-DIE

 

Selezionando la voce di menu destinatari SGA-DIE, l’utente visualizza la seguente maschera che permette di visualizzare, modificare o creare i riceventi delle SGA DIE.

 

 

Per effettuare la ricerca l'utente può utilizzare come filtri uno o più campi visualizzati all'interno della maschera. L’inserimento di più campi di ricerca determina la selezione d SGA DIE che soddisfano tutti i criteri inseriti (AND logico).

I campi visualizzati sono:

 

 

·          Edificio: conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una Pop Up che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “37” e selezionando il match code verranno presentati tutti gli edifici contenenti “3” e “7” nel proprio Codice Edificio, del tipo “037”, “337”, “370”, etc. e/o nel numero civico dell’indirizzo. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code corrispondente.

 

·          Email: è un campo di ricerca libera in cui inserire parte o tutta la email del destinatario che si sta cercando.

 

·          Abilitati a ricevere Email: selezionare un valore attraverso il match code per indicare se chi si ricerca è abilitato o no.

 

·           

Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

 

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

 

 

Selezionare il tasto  per visualizzare, nel frame in basso, tutte le richieste che soddisfano i criteri impostati. Per ognuna di esse vengono date le seguenti informazioni di dettaglio:

 

 

Selezionando l’icona  corrispondente alla descrizione del ricevente che si sta cercando, l’utente visualizza un pop up dove potrà apportare cambiamenti al destinatario.

 

Selezionando l’icona  corrispondente alla descrizione del ricevente che si sta cercando, l’utente cancellerà il destinatario.

 

Per creare un nuovo destinatario fare click su Nuovo che si trova sotto il pulsante per la ricerca.
